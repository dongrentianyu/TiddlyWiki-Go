## 新功能使用说明

### 表格视图

1. 点击视图切换按钮中的 "≡" 图标
2. 切换到表格视图模式
3. 点击列标题可以排序
4. 快速操作按钮在最后一列

### 开发者工具

1. 打开 Wiki 窗口后
2. 点击标题栏的开发者工具按钮 `<>`
3. 按照提示右键点击 Wiki 内容区域
4. 选择"检查"打开浏览器开发者工具

### 导出 HTML

1. 在 Wiki 卡片上点击"📦 导出 HTML"
2. 等待导出完成
3. 弹出确认对话框
4. 点击"确定"打开文件夹，"取消"仅关闭对话框

### 语言切换

1. 点击 TitleBar 上的语言切换按钮（EN/中）
2. 界面语言会切换（部分组件）
3. 设置会自动保存
